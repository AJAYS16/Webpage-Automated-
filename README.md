# CI/CD Pipeline Demo with GitHub, Jenkins, Docker, SonarQube & AWS

This is a sample project demonstrating a complete DevOps CI/CD pipeline using core tools like GitHub, Jenkins, Docker, SonarQube, and AWS.

---

## 🔧 Tech Stack

- **GitHub** – Version control
- **Jenkins** – CI/CD automation
- **Docker** – Containerization
- **SonarQube** – Code quality analysis
- **AWS** – Hosting (EC2, ECR, S3)
